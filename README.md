# Tanks Game - Telegram Mini App

Многопользовательская игра про танки для Telegram Mini App с экономикой в стиле Clash Royale.

## Особенности игры

- 🎮 20 уникальных танков 4-х редкостей
- 📈 Система прокачки до 13 уровня (+10% к характеристикам за уровень)
- 💰 Экономика с золотом и Telegram Stars
- 📦 Система сундуков (малый, средний, ультра)
- 🎯 Реалистичная механика стрельбы с разбросом
- 🤖 Умные боты с тактикой преследования
- 🗺️ 3 разные карты для боев
- 🎫 Система промокодов
